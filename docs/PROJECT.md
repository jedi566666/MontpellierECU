# Montpellier ECU (MTP)

## Overview

Montpellier ECU (MTP) is an open-source cryptocurrency project deployed on the Base network.

The project aims to create a transparent digital currency inspired by the historical identity of Montpellier and Occitanie, while remaining accessible worldwide.
