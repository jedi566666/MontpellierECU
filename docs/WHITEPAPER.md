# Montpellier ECU (MTP)
## White Paper

Version 1.0

---

# 1. Introduction

Montpellier ECU (MTP) is an open-source cryptocurrency project created with the ambition to build a transparent digital asset inspired by Montpellier and Occitanie heritage.

The project combines local identity with global accessibility through blockchain technology.

The Montpellier ECU is designed around transparency, public verification and community participation.

---

# 2. Vision

The vision of Montpellier ECU is to create a digital currency carrying the values of:

- Transparency
- Open source development
- Public blockchain verification
- Long-term community building

Born in Occitanie, designed for worldwide use.

"Née en Occitanie. Valable partout dans le monde."

# 4. Open Source Philosophy

The Montpellier ECU project follows an open-source approach.

Public resources include:

- Project documentation
- Logo and visual identity
- Token information
- Development history

The objective is to allow anyone to verify the project information.

# 5. Token Economy

The Montpellier ECU ecosystem is based on transparency.

Important principles:

- Public blockchain records
- Clear token supply information
- Transparent liquidity management
- Public documentation

Future token economy improvements will be documented through this repository.

---

# 6. Reserve & Value Philosophy

The project introduces the concept of a public reserve mechanism.

The reserve objective is to provide transparency regarding assets dedicated to supporting the ecosystem.

The difference between market price and fundamental project value remains clearly separated.

All reserve information must remain publicly verifiable.
